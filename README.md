# Helios DeFi Website

A dark luxury investment-style frontend for Helios DeFi with account registration, login, user dashboard, and admin panel.

## Updated Design

This version includes a more premium HYIP/investment-style homepage inspired by dark black/gold finance websites:

- Large luxury hero section
- Gold and black visual theme
- About Us company section
- Why Choose Us feature cards
- Investment plan cards
- Dashboard preview section
- Contact / call-to-action section
- Safer investment disclaimer wording

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Backend

The project is ready for Supabase. Use the SQL file inside:

```text
supabase/schema.sql
```

Create a `.env` file using `.env.example` when you connect your real Supabase project.

## Hosting Suggestion

Recommended low-cost setup:

- Domain: GoDaddy
- Hosting: Cloudflare Pages or Vercel
- Database/Auth: Supabase
